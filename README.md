# Temperature_Conversion_Project
## Overview

This is a **simple Python program** that converts temperatures between **Celsius, Fahrenheit, and Kelvin**.  
It is designed as a beginner-friendly project to practice **Python programming** and **using GitHub**.

---

## Features

- Convert Celsius to Fahrenheit or Kelvin  
- Convert Fahrenheit to Celsius or Kelvin  
- Convert Kelvin to Celsius or Fahrenheit  
- Command-line menu for easy interaction
